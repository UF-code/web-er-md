# Employee Record
Employee Record Application w/ Svelte, Routify, Axios, SvelteStrap, Material Icons

## Frontend Web
> Svelte, Routify, Axios Sveltestrap

## Backend
> Express, Cors, Dotenv, Mysql2, Sequelize

### Project Setup For Web & How To Compile n Run
```
npm install
npm run dev
```
